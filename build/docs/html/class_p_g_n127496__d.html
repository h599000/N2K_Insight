Constructor for PGN127496_d class, initializing data specific to PGN 127496.

The PGN127496_d class is derived from N2K_decomposed and represents the NMEA 2000 message with PGN 127496, which typically contains information about the engine parameters like time to empty, distance to empty, estimated fuel remaining, and trip run time.

Parameters
TimeOfReceiptThe time at which the message was received
PgnThe PGN of this NMEA 2000 message.
TimetoEmptyThe calculated time remaining before the fuel tank is empty, based on current usage.
DistancetoEmptyThe calculated distance that can be covered before the fuel tank is empty, based on current usage.
EstimatedFuelRemainingAn estimate of the amount of fuel remaining in the tank.
TripRunTimeThe cumulative time the trip has been running.
