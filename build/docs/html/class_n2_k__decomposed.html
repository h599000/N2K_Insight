Constructor for the N2K_decomposed class, which initializes an instance with given parameters.

This constructor initializes an N2K_decomposed object with the specific time of receipt and Parameter Group Number (PGN). These values are used to represent the metadata associated with a decomposed NMEA 2000 message.

Parameters
TimeOfReceiptThe time at which the message was received.
PgnThe PGN of this NMEA 2000 message.
